# CANTARE
This repository serves as an **updated** and GitHub workflow to produce multi-omic network-based predictive models using the "CANTARE" software. Refer to the citation at the bottom of the repository for author and publication information.













### Citation

*This analysis used a modified version of the "CANTARE" software (Siebert, 2021), which is available through the publication and on SourceForge at [https://sourceforge.net/projects/cytomelodics/](url).*

Publication: 
Siebert JC, Saint-Cyr M, Borengasser SJ, Wagner BD, Lozupone CA, Görg C. CANTARE: finding and visualizing network-based multi-omic predictive models. BMC Bioinformatics. 2021 Feb 19;22(1):80. doi: 10.1186/s12859-021-04016-8. PMID: 33607938; PMCID: PMC7896366.
